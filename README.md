# 🌍 Wanderlust – Full-Stack Hotel Listing App

## 📌 About the Project
**Wanderlust** is a full-stack **hotel listing and booking web application** inspired by Airbnb.  
It allows users to explore, filter, and view hotel details with a responsive and intuitive interface.

---

## ⚙️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  

---

## 🚀 Features
- 🔹 Add, view, and manage hotel listings  
- 🔹 Filter and sort hotels by category or price  
- 🔹 Responsive and user-friendly UI  
- 🔹 Integrated backend with MongoDB for data storage  

---

## 🧠 Learning Highlights
- Practiced **RESTful API** design using Express.js  
- Implemented **CRUD operations** with MongoDB  
- Focused on **responsive design** and clean UI layout
